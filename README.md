# Определитель типа треугольника  
**WinForms приложение для классификации треугольника по длинам сторон**

---

## О проекте

Этот проект создан в рамках лабораторной работы по МДК 01.02 «Поддержка и тестирование программных модулей».  
Программа позволяет **определить тип треугольника** *(равносторонний, равнобедренный, разносторонний)* на основе введённых длин сторон, а также выполняет **валидацию входных данных** согласно теореме существования треугольника.

*«Цель: создать программу, которая определяет вид треугольника и протестировать его по различным характеристикам.»*

### Основные элементы тестирования

| Приоритет   | Элемент тестирования        | Описание                                  |
|-------------|-----------------------------|-------------------------------------------|
| Критический | Корректность определения типа | Правильная работа основной функции        |

### Соответствие видов тестирования

| Элемент тестирования      | Вид тестирования       | Метод отслеживания           |

| Защита от некорректного ввода | Функциональное     | Тест-кейсы, покрытие кода    |

| Удобство использования    | UX тестирование        | Опрос, чек-лист              |
| Обработка исключений      | White-box              | Покрытие кода, тест-кейсы    |
| Граничные значения        | Black-box              | Тест-кейсы                   |

## Фрагмент кода (метод определения треугольника)

```csharp
public void opredelenie()
{
    try
    {
        // Получаем значения из текстовых полей
        int a = Convert.ToInt32(textBox1.Text);
        int b = Convert.ToInt32(textBox2.Text);
        int c = Convert.ToInt32(textBox3.Text);

        // Проверка на отрицательные числа
        if (a <= 0 || b <= 0 || c <= 0)
        {
            label5.Text = "Стороны не могут быть отрицательными или равны нулю!";
            return;
        }

        // Проверка теоремы треугольника
        if (a + b > c && a + c > b && b + c > a)
        {
            if (a == b && b == c)
                label5.Text = "Равносторонний";
            else if (a == b || a == c || b == c)
                label5.Text = "Равнобедренный";
            else
                label5.Text = "Разносторонний";
        }
        else
        {
            label5.Text = "Не соответствует теореме о существовании треугольника";
        }
    }
    catch (FormatException)
    {
        label5.Text = "Ошибка: Введите корректные числа!";
    }
    catch (Exception ex)
    {
        label5.Text = $"Ошибка: {ex.Message}";
    }
}
